Webゲーム制作講習会用のページ

講習会の概要

この講習会は、WebページやWebゲームを制作するハンズオン形式のイベントです。

ゲーム班に所属していない方でも、電気技術部の部員であればどなたでもご参加いただけます。

対象者

  • プログラミングに興味がある方
  • Web制作に興味がある方
  • 環境構築などの複雑な作業を避けて、まずは何かを作ってみたい方
  • ハッカソンなどのイベントに参加してみたい方
  • その他、興味のある方も歓迎します

事前準備

今回の講習会では、環境構築なしで作業を始められる GitHub Codespaces を使用します。

GitHubアカウントの作成
以下のリンクからアカウントを作成してください：
・https://github.com/signup
登録後は、メール認証も完了させておいてください。
使用するブラウザ
Google Chrome または Microsoft Edge を推奨します。
